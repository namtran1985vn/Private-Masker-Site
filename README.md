## About Private Masker

Private Masker is an iOS app that uses on-device AI to:
- Automatically detect and blur faces in videos
- Remove metadata for untraceable sharing
- Modify voice pitch for additional privacy
- Export in resolutions from 480p to 4K

**Key Privacy Promise:** All processing happens locally on your device. No uploads, no cloud processing, no data collection.
